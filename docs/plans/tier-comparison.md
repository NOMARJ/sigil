## Tier Details

### 🚫 Anonymous (No Account)
- **For:** Quick one-off scans
- **Includes:** Basic static analysis only (Phases 1-8)
- **Limitations:** No AI features, no scan history, no saved reports

### 🆓 Free (Logged In)
- **For:** Individual developers trying Sigil
- **50 credits/month** = ~50 basic scans
- **Requires:** Email registration
- **Includes:** Scan history, basic dashboards
- **Upgrade prompt:** After using 40/50 credits

### 💼 Pro ($29/month)
- **For:** Professional developers and security engineers
- **5,000 credits/month** = ~625 scans or 250 deep investigations
- **Unlocks:** 
  - Full AI-powered analysis
  - Interactive chat about findings
  - Export reports as PDF/JSON
  - Email alerts for critical findings

### ⚡ Elite ($79/month)
- **For:** Power users, security researchers, DevSecOps teams
- **15,000 credits/month** = ~1,875 scans or 750 investigations
- **Everything in Pro, plus:**
  - Claude Sonnet access for complex analysis
  - API access for CI/CD integration
  - Custom detection rules
  - Priority support response
  - Bulk scanning tools
  - Advanced analytics dashboard

### 👥 Team ($199/month)
- **For:** Development teams and small companies
- **50,000 shared credits** across 5 seats
- **Everything in Elite, plus:**
  - Team management dashboard
  - Shared scan results
  - Role-based access control
  - Usage analytics per member
  - SSO integration
  - Slack/Discord webhooks

### 🏢 Enterprise (Custom)
- **For:** Large organizations with compliance needs
- **Unlimited credits**
- **Everything in Team, plus:**
  - Unlimited seats
  - On-premise deployment option
  - Custom AI model fine-tuning
  - SLA guarantees
  - Dedicated support engineer
  - SOC2/ISO compliance reports

## Why "Elite" Instead of "Pro+"?

We chose **Elite** over alternatives because:
- **Clear positioning:** Sits naturally between Pro and Team
- **Aspirational:** Appeals to power users who want the best
- **No confusion:** Avoids the "+" naming that feels like an afterthought
- **Gaming familiar:** Resonates with developers (Elite dangerous, Elite hacker)

Other names considered:
- ~~Pro+~~ - Feels like an add-on, not a distinct tier
- ~~Premium~~ - Overused, doesn't differentiate from Pro
- ~~Advanced~~ - Too technical, less aspirational
- ~~Expert~~ - Implies skill requirement, might discourage upgrades
- ~~Master~~ - Sounds like a certification level

## Credit Usage Examples

### Typical Elite User (15,000 credits/month)
- **Daily security scans:** 30 scans × 8 credits × 30 days = 7,200 credits
- **Weekly deep dives:** 4 investigations × 32 credits × 4 weeks = 512 credits
- **Interactive sessions:** 50 Q&As × 2 credits = 100 credits
- **Remaining buffer:** 7,188 credits for experiments and API calls

### When to Upgrade

**Free → Pro**
- Running out of credits by day 5
- Need AI analysis for production code
- Want to investigate false positives

**Pro → Elite**
- Using >4,000 credits monthly
- Need API access for automation
- Want Claude Sonnet for complex code
- Require priority support

**Elite → Team**
- Multiple developers need access
- Want centralized billing
- Need audit logs for compliance
- Require SSO/SAML
