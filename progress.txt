## Codebase Patterns

- API services use credit_service for all LLM operations
- Frontend components fetch credit balance on mount
- Use Claude Haiku for simple tasks, Sonnet for complex analysis
- All interactive features deduct credits atomically via stored procedures
- LLM prompts are built using template functions for consistency
