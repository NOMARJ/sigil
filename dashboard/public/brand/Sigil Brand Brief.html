Sigil · Brand brief · v1.0

Sigil identity system.

Two marks, one family. The Brace is the primary identity — the brand you see at every product surface. The Seal is the verification artefact — a sister mark that signs scanned packages.

01 · Principles

What the system has to do, and what it must never do.

001

Read as quarantine, not decoration.

The Brace is a containment vessel — brackets sealing an unknown. Sigil's job is to hold something potentially hostile in place while we look at it. The mark must communicate that pause.

002

Built for terminals, not just websites.

The mark must hold its meaning at 16px in a CLI line, in a README badge, and on a browser favicon. If it doesn't survive the smallest surface, it doesn't ship.

003

The bind-rune is the soul.

The vertical stave with four branches is a protective seal — historically, a mark scratched onto a thing that needed warding. It's the one element that ties every artefact in the system together.

004

Verdicts have shape, not just colour.

Clean, Quarantine, and High-risk states share the Seal silhouette but shift colour. Colour-blind users still get a recognisable verification artefact; sighted users get an instant verdict.

03 · Wordmark & lockups

JetBrains Mono Bold, all caps, locked tracking. The wordmark is a typographic asset, not a freeform setting — use the lockup files; do not retype.

06 · Verdict family

08 · Do / Don't

Hard rules. The system breaks fast if these slip.

✓ Do
  • Use the supplied SVG files. Don't regenerate or trace.
  • Pair the Seal with a text verdict label, always.
  • Use Brace below 32px; switch Seal to sigil-seal-small.svg.
  • Maintain clear space equal to the inner bracket gap.
  • Pin the wordmark to JetBrains Mono Bold, all caps.
× Don't
  • Don't recolour the rune separately from the bracket.
  • Don't apply gradients, glows, drop-shadows, or strokes.
  • Don't rotate, skew, or stretch either mark.
  • Don't replace the bind-rune with a checkmark or padlock.
  • Don't use the Seal as a primary identity — that's Brace's job.
  • Don't use Brace as a verdict — that's Seal's job.

React — component wrapper

import Brace from './brand/brace/sigil-brace-currentcolor.svg';
import Seal  from './brand/seal/sigil-seal-currentcolor.svg';

export const SigilMark = ({ size = 24, color = 'currentColor' }) =>
  <span style={{ color, display: 'inline-flex' }}>
    <Brace width={size} height={size} />
  </span>;

export const VerdictSeal = ({ verdict, size = 24 }) => {
  const src = {
    clean:      '/brand/seal/sigil-seal-clean.svg',
    quarantine: '/brand/seal/sigil-seal-quarantine.svg',
    risk:       '/brand/seal/sigil-seal-risk.svg',
  }[verdict];
  return <img src={src} width={size} height={size} alt={verdict} />;
};
