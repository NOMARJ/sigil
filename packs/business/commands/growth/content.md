# /growth:content

Produce content strategy or a specific content piece.

## Usage

```
/growth:content <product>                  # content strategy for this product
/growth:content <product> post             # one blog post (AEO-optimised)
/growth:content <product> linkedin         # one LinkedIn post
/growth:content <product> readme           # GitHub README review/rewrite
/growth:content <product> hn               # Hacker News post draft
/growth:content <product> icp             # ICP document
/growth:content <product> calendar        # 4-week content calendar
```

## What it does

Reads `content-engine` skill (and `icp-builder` for icp option).
Calibrates output to the product's GTM motion.
Every piece includes AEO optimisation checklist.

## For InstaIndex specifically

Every post targets an explicit question that someone would ask an AI search engine.
Title format: question or direct answer. First paragraph: answer the question completely.
